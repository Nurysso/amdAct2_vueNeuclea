## Extension Points

The following capabilities are **IR-supported but not yet implemented** in the generated output:

- **Authentication**: `ServiceSpec.securitySchemes` carries upstream auth metadata. Inject headers in `src/http-client.ts` and `MCP_API_KEY` / `MCP_BEARER_TOKEN` env vars in `src/config.ts`.
- **SSE transport**: Replace `StdioServerTransport` with the SDK's SSE transport. The `TransportKind` field in the IR already models this.
- **HTTP transport**: Same swap point as SSE; the IR `transport` field is `"http"`-ready.
